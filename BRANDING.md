# Guide de Marque - POP (Plateforme d'Outils Pédagogiques)

Ce document sert de référence officielle pour l'identité visuelle de la plateforme POP. **Ne pas réinventer ces éléments lors de futurs ajouts.**

## 1. Identité Nominale
- **Nom Officiel** : POP
- **Acronyme de** : Plateforme d'Outils Pédagogiques
- **Slogan** : Booster votre enseignement

## 2. Logo Officiel
- **Fichier** : `/public/icons/brain-logo.png`
- **Description** : Cerveau neuronal brillant dans une capsule circulaire.
- **Style** : Verre poli (*glassmorphism*), lueur interne orange néon.
- **Usage** : Utilisé dans la Super-Tuile Hero et comme icône de base de la plateforme.

## 3. Charte Graphique
- **Couleur Primaire** : `#f6921e` (Orange POP)
- **Couleur Secondaire** : `#10b981` (Vert émeraude - utilisé pour l'interactivité)
- **Arrière-plan** : Sombre profond (`#0f111a`) avec gradients radiaux.

## 4. Typographie
- **Titres / Logo** : `Orbitron` (Poids : Black/ExtraBold) - Look futuriste/technologique.
- **Corps de texte** : `Inter` (Sans-serif) - Pour une lisibilité maximale.

## 5. UI / UX
- **Style global** : Glassmorphism (panneaux translucides, flou d'arrière-plan, bordures fines).
- **Interactions** : Effet `active-glow` (lueur orange) et légère mise à l'échelle (`scale`) au survol.
